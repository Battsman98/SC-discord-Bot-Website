# Chrome Web Store listing

## Name

Game Assist RSI Hangar Importer

## Summary

Import only your pledged Star Citizen ships and vehicles into SC Companion.

## Category

Productivity (or Games, if offered by the dashboard)

## Detailed description

Game Assist RSI Hangar Importer is a narrowly scoped companion for the SC Companion website.

After you click Import RSI Hangar on SC Companion, the extension uses your existing signed-in RSI browser session to request your My Hangar pledge pages. It processes those responses locally and sends only recognized ship and vehicle names to SC Companion.

The importer supports standalone ships, game-package vehicles, and ships contained in upgraded pledges. It excludes CCUs, paints, equipment, weapons, armor, flair, currency, coupons, gift cards, and other non-vehicle extras.

The extension does not ask for, display, store, or transmit your RSI password or session-cookie value. It cannot purchase, gift, exchange, reclaim, melt, or modify pledges. It performs no background polling and accesses RSI only after a user starts an import.

SC Companion uses imported names to build a private hangar with responsive ship cards, ownership labels, loaners, quantities, and ship details.

This is an unofficial fan-made companion. It is not affiliated with, endorsed by, sponsored by, or authorized by Cloud Imperium Games or Roberts Space Industries, and it does not claim authorization from either company.
