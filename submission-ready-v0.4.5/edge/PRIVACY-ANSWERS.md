# Microsoft Edge Privacy answers

## Single purpose

Import the names of ships and vehicles in a user's own RSI pledge hangar into the user's authenticated SC Companion hangar.

## Permissions

`https://robertsspaceindustries.com/*` permits read-only requests to the signed-in user's RSI pledge pages after the user starts an import. `https://sccompanion.org/*` permits the production website to initiate the import and receive normalized ship/vehicle names. Messages from other origins are rejected.

No cookies, tabs, browsing-history, storage, downloads, webRequest, scripting, or optional permissions are requested.

## Remote code

No remote code is downloaded or executed. RSI responses are data only.

## Data usage

RSI page responses are transiently processed locally. Only recognized ship and vehicle names are transmitted to SC Companion. Complete HTML, passwords, session-cookie values, billing information, order information, pledge IDs, buybacks, CCUs, paints, equipment, and unrelated browsing data are not transmitted. Data is not sold or used for advertising, analytics, credit decisions, or unrelated purposes.

Privacy policy: https://sccompanion.org/rsi-hangar-importer/privacy
