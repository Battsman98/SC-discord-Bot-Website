# Microsoft Edge Add-ons listing

## Name

Game Assist RSI Hangar Importer

## Short description

Import only your pledged Star Citizen ships and vehicles into SC Companion.

## Description

Game Assist RSI Hangar Importer connects the SC Companion website to the signed-in user's RSI My Hangar pages.

After the user clicks Import RSI Hangar, pledge responses are processed locally. Only recognized ship and vehicle names are sent to SC Companion. Standalone, package, and upgraded-pledge ships are supported. CCUs, paints, equipment, weapons, armor, flair, currency, coupons, gift cards, and other extras are excluded.

The extension does not ask for, display, store, or transmit an RSI password or session-cookie value. It cannot purchase, gift, exchange, reclaim, melt, or modify pledges, and it performs no background polling.

This is an unofficial fan-made companion. It is not affiliated with, endorsed by, sponsored by, or authorized by Cloud Imperium Games or Roberts Space Industries, and it does not claim authorization from either company.
